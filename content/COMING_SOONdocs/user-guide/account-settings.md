## Profile settings

Navigate to **Settings → Profile** to update your display name, timezone, and default trading style. Timezone affects how timestamps are displayed throughout the journal and intel feed.

## Data sources

Connect market data providers under **Settings → Data Sources**. Ampliterm supports multiple provider adapters. Configure API keys for each provider you want to enable.

## Appearance

The terminal is dark-mode only by design. You can adjust the font size and information density under **Settings → Appearance**.

## Export & backup

Export your full journal and simulation history as a JSON archive from **Settings → Export**. This archive can be imported into a fresh Ampliterm instance to migrate your data.
