## Global

| Shortcut | Action |
|----------|--------|
| `⌘K` | Open command palette / search |
| `⌘N` or `N` | New journal entry |
| `⌘S` | Save current entry |
| `G J` | Go to Journal |
| `G S` | Go to Simulations |
| `G I` | Go to Intel Feed |
| `G M` | Go to Markets |
| `?` | Show keyboard shortcut overlay |
| `Esc` | Close modal / dismiss overlay |

## Journal

| Shortcut | Action |
|----------|--------|
| `E` | Edit selected entry |
| `R` | Run simulation on selected entry |
| `⌘↑` / `⌘↓` | Navigate between entries |
| `F` | Filter panel |
| `T` | Tag selected entry |

## Simulations

| Shortcut | Action |
|----------|--------|
| `↵` | Run simulation |
| `C` | Compare selected simulations |
| `⌘E` | Export results as CSV |
