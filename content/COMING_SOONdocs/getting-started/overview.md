## What is Ampliterm?

Ampliterm is a **research-first trading terminal** designed for systematic traders, quants, and serious investors. It combines four core modules into a single self-hosted workspace.

### Journal

Log trades with rich metadata — thesis, tags, screenshots, and linked documents. Every entry is time-stamped, searchable, and linked to simulations. Your research layer, structured and permanent.

### Simulations

Backtest strategy hypotheses against historical data. Run Monte Carlo scenarios, view distribution of outcomes, and compare simulation runs side-by-side before committing real capital.

### Intel Feed

Aggregated market signals from earnings, macro events, unusual options flow, and analyst rating changes — filtered to your watchlist in real time.

### API

Programmatic access to your journal, simulation results, and intel feed. Build custom dashboards, automate logging, or integrate with your broker.

## Prerequisites

- A running Ampliterm instance (self-hosted on your own server)
- A modern browser: Chrome 120+, Firefox 121+, or Safari 17+
- For API access: your instance's API endpoint and a valid API key
